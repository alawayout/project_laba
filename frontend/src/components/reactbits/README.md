# reactbits components

Components sourced from https://reactbits.dev — copy the component source here.

Recommended for LABBOR STUDIO:
- `AnimatedList` — menu animations, toast queue, stage list
- `CountUp` — StatItem big numbers
- `SpotlightCard` / `TiltedCard` — OrderCard hover effect
- `AnimatedContent` — Modal pop-in entrance
- `Dock` / `GooeyNav` — NavRail (optional, if animated nav is needed)

Usage: copy the component from reactbits.dev into this folder, then import normally:
```ts
import { CountUp } from "@/components/reactbits/CountUp";
```
