# Соответствие иконок: Figma → lucide-react

Иконки из макета не импортируются. Ниже — используемые аналоги из `lucide-react`.

| Назначение в UI | Компонент lucide | Где используется |
|---|---|---|
| Главная / Наряды | `LayoutGrid` | NavRail |
| Аналитика | `LineChart` | NavRail |
| Зарплаты | `Wallet` | NavRail |
| Виды работ | `FileText` | NavRail |
| Документы | `FolderOpen` | NavRail |
| Поиск | `Search` | SearchInput |
| Календарь / дата | `Calendar` | DatePill |
| Уведомления | `Bell` | NotificationBell |
| Назад | `ArrowLeft` | OrderDetailView |
| На главную | `Home` | OrderDetailView |
| Закрыть (модалка) | `X` | Modal |
| Поле «только чтение» | `Lock` | Field, Toggle (бегунок) |
| Замена техника | `Repeat` | StageRow, Badge (clinic) |
| Штраф / дедлайн / внимание | `TriangleAlert` | StageRow, Badge (dead), Toast (warn) |
| Отчёт по этапу | `ClipboardList` | StageRow |
| Готово / выполнено | `Check` | Badge (done), Toast (ok) |
| Рост показателя | `ArrowUp` | StatItem |
| Раскрытие select | `ChevronDown` | Select |

Размер по умолчанию — 24px, толщина обводки 1.8–2.0 (терминальные статусы — 2.4).
